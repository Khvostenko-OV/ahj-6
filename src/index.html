Collapse
1. Важно: в рамках этого ДЗ можно использовать любой менеджер пакетов.
2. Вы можете сделать все три задачи в одном репозитории и показать работу всех 
трёх виджетов на одной странице. Главное, чтобы все виджеты были оформлены в виде 
отдельных классов.
3. Всё, включая картинки и стили, должно собираться через Webpack и выкладываться 
на GitHub Pages через AppVeyor.
4. В README.md должен быть размещён бейджик сборки и ссылка на GitHub Pages.
5. В качестве результата пришлите проверяющему ссылки на ваши GitHub-проекты.
6. Автотесты писать не требуется.
